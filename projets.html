Koo2Fourchette est un projet de site web culinaire
permettant aux utilisateurs de consulter, partager et
proposer des recettes de cuisine variées.


Objecrifs du projet :

  • Concevoir une maquette HTML/CSS fidèle à une charte donnée.
  • Programmer les fonctionnalités du site (recherche, gestion des des recettes etc.)
    dans un langage de programmation web (par exemple PHP, Pyhton ou JavaScript).
  • Héberger le site sur une machine virtuelle (VM) pour simuler un déploi
